Amp Lab Speaker Design

I joined the AMP Lab as soon as I knew what it was. I was so interested in being able to actually start building/designing electronics instead of learning how to theoretically analyse them.

I'm a music geek, so I felt like I wanted to learn about the science behind sound while designing a speaker. My plan is to start by making this a wireless bluetooth speaker and morphing it into the center speaker of a wireless surround sound system

Sadly, due to covid, this project will remain in the design phase until further notice. I will be continuing work on this project, it is not retired!


Xbox 360 Kinect Programming

I have heard about people using the sensors on a kinect before to do a multitude of different programs. This really interested me, so I wanted to try this myself.

I want to learn to use the kinect to learn about object tracking and to figure out how to use the data it collects via the IR sensor.

To do this, I would like to create a game, object tracking and learning how to manipulate objects and other variables via the tracking, and create a program to turn the IR sensor data into a 3D model of an object.

Currently, I am waiting to recieve PC adapters for the kinect.
